# Employee Management SQL Project

This is a MySQL Employee Management project created in phpMyAdmin.

## 📌 Project Objective
The goal of this project is to manage employee records using SQL.

## 🛠 Tools Used
- MySQL
- phpMyAdmin
- AWebServer
- GitHub

## 📂 Database Name
employee_management

## 📋 SQL Concepts Used
- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- UPDATE
- DELETE
- SELECT
- WHERE
- ORDER BY
- VIEWS
- STORED PROCEDURES

## 📊 Features
- Add employee records
- Update employee details
- Delete employee records
- View all employees
- Search employees by department
- Use views for reports
